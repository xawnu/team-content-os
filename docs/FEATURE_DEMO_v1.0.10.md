# 参考视频池可视化 - 功能演示

## 🎬 核心功能展示

### 1. 添加视频
![添加视频](https://via.placeholder.com/800x400/0066cc/ffffff?text=1.+粘贴+YouTube+链接+→+点击添加+→+自动显示视频卡片)

**操作步骤：**
1. 在输入框粘贴 YouTube 视频链接
2. 点击"添加"按钮（或按回车键）
3. 系统自动解析视频信息
4. 显示包含缩略图、标题、时长的视频卡片

**支持的链接格式：**
- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/embed/VIDEO_ID`
- 直接输入 11 位视频 ID

---

### 2. 拖拽排序
![拖拽排序](https://via.placeholder.com/800x400/28a745/ffffff?text=2.+鼠标拖拽+≡+图标+→+调整视频顺序)

**操作步骤：**
1. 鼠标悬停在视频卡片左侧的 `≡` 图标
2. 按住鼠标左键拖拽
3. 移动到目标位置后松开
4. 视频顺序立即更新

**为什么顺序重要？**
- 排在前面的视频权重更高
- 生成文案时会优先参考前面的视频风格
- 建议把最想模仿的视频放在第一位

---

### 3. 快速删除
![快速删除](https://via.placeholder.com/800x400/dc3545/ffffff?text=3.+鼠标悬停+→+点击+×+按钮+→+删除视频)

**操作步骤：**
1. 鼠标悬停在视频卡片上
2. 右上角出现红色 `×` 按钮
3. 点击按钮即可删除
4. 无需确认，立即生效

---

### 4. 空状态提示
![空状态](https://via.placeholder.com/800x400/6c757d/ffffff?text=4.+空状态+→+友好提示+→+引导添加视频)

**显示内容：**
- 大图标（视频播放器）
- "还没有添加参考视频"
- "至少添加 1 个视频才能生成文案"

---

### 5. 自动保存
![自动保存](https://via.placeholder.com/800x400/17a2b8/ffffff?text=5.+添加/删除/排序+→+自动保存到+localStorage)

**特性：**
- 无需手动保存
- 刷新页面后自动恢复
- 支持按主题保存多个参考池
- 跨浏览器标签页同步

## 🎯 完整使用流程

### 场景：生成一篇关于"家庭养鸡"的视频文案

1. **添加参考视频**
   ```
   输入框：https://youtube.com/watch?v=abc123
   点击"添加" → 显示视频卡片
   
   [缩略图] How to Build a Chicken Coop
   Homestead Roots · 12:34
   ```

2. **继续添加第二个视频**
   ```
   输入框：https://youtu.be/def456
   点击"添加" → 显示第二个卡片
   
   [缩略图] Raising Chickens for Beginners
   Farm Life · 8:45
   ```

3. **调整顺序**（如果第二个视频更重要）
   ```
   拖拽第二个卡片到第一位
   
   ✓ Raising Chickens for Beginners (现在是第一位)
   ✓ How to Build a Chicken Coop (现在是第二位)
   ```

4. **填写其他参数**
   ```
   参考频道：@homesteadrootss
   输出方向：同类型视频详细文案
   主题锁定：chicken coop
   内容目的：拉新破圈
   叙事结构：问题→方案→结果
   ```

5. **生成文案**
   ```
   点击"生成 1 篇详细文案"按钮
   等待 10-30 秒
   查看生成结果
   ```

## 💡 使用技巧

### 技巧 1：选择高质量参考视频
- ✅ 选择播放量高、评论多的视频
- ✅ 选择结构清晰、节奏明快的视频
- ✅ 选择与目标主题高度相关的视频
- ❌ 避免选择过长（>20分钟）的视频
- ❌ 避免选择纯音乐或无口播的视频

### 技巧 2：合理安排顺序
- 第一位：最想模仿的视频风格
- 第二位：补充的内容角度
- 第三位：差异化的表达方式

### 技巧 3：保存主题参考池
```
主题名称：homestead-chicken
点击"保存该主题参考池"
下次直接加载，无需重新添加
```

### 技巧 4：快速测试不同组合
```
方案 A：3 个教程类视频
方案 B：2 个教程 + 1 个 Vlog
方案 C：1 个深度教程 + 2 个快速技巧

分别生成，对比效果
```

## 🔧 故障排查

### 问题 1：点击"添加"后没有反应
**可能原因：**
- YouTube API Key 未配置
- API 配额已用完
- 视频链接格式错误

**解决方法：**
1. 检查 `.env.local` 中的 `YOUTUBE_API_KEY`
2. 查看浏览器控制台错误信息
3. 尝试使用不同格式的链接

### 问题 2：视频卡片显示不完整
**可能原因：**
- 网络问题导致缩略图加载失败
- 视频已被删除或设为私密

**解决方法：**
1. 刷新页面重试
2. 检查视频是否可以在 YouTube 正常访问
3. 更换其他视频

### 问题 3：拖拽不生效
**可能原因：**
- 浏览器不支持拖拽 API
- 鼠标没有按住 `≡` 图标

**解决方法：**
1. 确保鼠标按住左侧的 `≡` 图标
2. 尝试使用 Chrome/Edge/Firefox 最新版本
3. 如果仍然不行，可以删除后重新添加来调整顺序

### 问题 4：刷新后视频消失
**可能原因：**
- localStorage 被清空
- 浏览器隐私模式

**解决方法：**
1. 不要使用隐私/无痕模式
2. 检查浏览器是否允许 localStorage
3. 使用"保存该主题参考池"功能备份

## 🎓 最佳实践

### 1. 参考视频数量
- **最少**：1 个（必需）
- **推荐**：2-3 个（平衡质量和多样性）
- **最多**：3 个（系统限制）

### 2. 视频时长
- **推荐**：5-15 分钟
- **可接受**：3-20 分钟
- **不推荐**：<3 分钟或 >20 分钟

### 3. 视频类型
- ✅ 教程类（How-to）
- ✅ 清单类（Top 10）
- ✅ 对比类（A vs B）
- ✅ 挑战类（Challenge）
- ❌ 纯音乐视频
- ❌ 无口播的 Vlog

### 4. 更新频率
- 每周更新一次参考池
- 关注最新的高增长视频
- 定期清理过时的参考视频

## 🚀 进阶玩法

